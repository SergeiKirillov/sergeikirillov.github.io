Монитор производительности¶

  1. Запустите «Монитор производительности» командой perfmon.
  2. Перейдите в раздел «Группы сборщиков данных» → «Системные» → «System Performance».
  3. Кликните по подразделу правой кнопкой мыши и выберите «Пуск». Перейдите в раздел «Отчеты» → «Системные» → «System Performance» и выберите последний отчет. Обратите внимание на результаты проверки. Если в ходе мониторинга будет выявлена проблема, она отобразится в блоке «Предупреждения». Обратите внимание на результаты проверки. Если в ходе мониторинга будет выявлена проблема, она отобразится в блоке «Предупреждения». В нашем примере замедление системы связано с недостатком оперативной памяти. Здесь же указан симптом проблемы — слишком интенсивный страничный обмен. Для получения дополнительных сведений рекомендуется изучить другие разделы отчета, используя Gemini или аналогичную нейросеть для анализа данных.

Проверка оперативной памяти¶

Для анализа работы оперативной памяти можно использовать встроенную утилиту mdsched. Запустите ее командой mdsched.exe и подтвердите перезагрузку компьютера. Отчет также доступен в журнале событий Windows: «Журналы Windows» → «Система», событие MemoryDiagnostics-Results.

Проверка скорости диска¶

Для оценки производительности жесткого диска можно использовать команду winsat disk — выполните ее в запущенной от имени администратора командной строке или PowerShell. Менее чем за минуту получите отчет — обратите внимание на следующие показатели:

  • Disk Sequential 64.0 Read — скорость чтения.
  • Disk Sequential 64.0 Write — скорость записи.
  • Disk Random 16.0 Read — скорость чтения небольших блоков данных в рандомных частях диска.
  • Задержка — отклик диска.
  • Максимальное значение — подвисания диска.

Результаты тестирования нашего диска с интерфейсом SATA III весьма хорошие: 469,01 МБ/с при чтении и 489,51 МБ/с при записи.

Средняя задержка 2,046 мс указывает на отличную производительность, а максимальное значение 156,132 мс также находится в пределах нормы, хотя в идеале оно могло бы быть ниже. Мой Домашний ПК
